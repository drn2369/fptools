# operation_swiper
Swiper no swiping

## Building

Required tools:
* Cmake
* Git
* c/c++ compiler
* libeigen3-dev

To build the source:

```
	$ git clone https://github.com/CaseWallet/operation_swiper.git
	$ cd operation_swiper
	$ mkdir build
	$ cd build
	$ cmake ../
	$ make
```
To build the doc:
```
    $ make doc
```
